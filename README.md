A lightweight and high-performance price tracking system that monitors product price changes and sends instant notifications through a Telegram Bot.

Tech Stack:
Backend: C++ 20, Drogon Framework (High-performance non-blocking I/O).
Database: PostgreSQL.
Interface: Telegram Bot API.

Core Features
Real-time Price Tracking: Monitor specific product prices from various websites by simply providing a URL.
